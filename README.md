- `main.py` — входная точка.
- `game.py` — игровой цикл, меню, переключение состояний.
- `settings.py` — размеры, цвета, баланс.
- `models.py` — небольшие структуры данных.
- `helpers.py` — функции для отрисовки, тайлов и звёзд.
- `level.py` — логика комнаты: двери, охранники, выстрелы, выход.
- `levels.py` — описание уровней и проверка важных клеток.
- `entities.py` — игрок, эхо, охранники.
- `objects.py` — кнопки, двери, монеты.
- `map_tools.py` — карта и A*.
- `record_system.py` — запись текущего забега.
- `save_manager.py` — сохранение рекордов и настроек.
- `sound_manager.py` — генерация и проигрывание звуков.
- `particles.py` — простые визуальные эффекты.
